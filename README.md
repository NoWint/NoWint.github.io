## 🌐 在线预览

访问 GitHub Pages 链接查看（部署后）：`https://你的用户名.github.io/仓库名`

```
MyPage/
├── index.html              # 主页面
├── glasswing-hero-video.webm  # Hero 视频资源

```

## 🚀 部署到 GitHub Pages

### 方法一：使用 Git 命令行

```bash
# 1. 初始化 Git 仓库（如果还没有）
git init

# 2. 添加所有文件
git add .

# 3. 提交
git commit -m "Initial commit: Project Nowint website"

# 4. 关联远程仓库（替换为你的仓库地址）
git remote add origin https://github.com/你的用户名/你的仓库名.git

# 5. 推送
git push -u origin main
```

### 方法二：使用 GitHub Desktop

1. 打开 GitHub Desktop
2. 添加本地仓库（Add Local Repository）
3. 选择此项目文件夹
4. 发布到 GitHub（Publish to GitHub）

### 方法三：直接上传文件

1. 在 GitHub 上创建新仓库
2. 点击 "uploading an existing file"
3. 拖拽所有文件上传
4. 提交 commit

## ⚙️ GitHub Pages 设置

1. 进入仓库的 **Settings**
2. 左侧菜单选择 **Pages**
3. Source 选择 **Deploy from a branch**
4. Branch 选择 **main** (或 master)，文件夹选择 **/(root)**
5. 点击 **Save**

等待几分钟后，你的网站就会在以下地址生效：
```
https://你的用户名.github.io/仓库名/
```
